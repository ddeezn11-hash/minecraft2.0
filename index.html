LIFETIME MULTIPLAYER SETUP
To run a permanent server your friends can always join:
Option 1 — Free (Render.com)
1. Create account at render.com
2. New → Web Service → paste the server code (ENGINE → SERVER tab)
3. Set start command: node server.js
4. Get your wss://yourapp.onrender.com URL — share it!

Option 2 — Glitch.com (always-on free)
1. glitch.com → New Project → paste server code
2. Your URL: wss://yourproject.glitch.me

Option 3 — Your own machine
1. Run server.js locally (node server.js)
2. Use ngrok: ngrok tcp 8080 → share the URL
3. For truly permanent: forward port 8080 in router settings

wss:// vs ws://
Use wss:// (secure) when your game is served over HTTPS.
